### 🧬 [TCGA Lung Cancer Multi-Omics Analysis](https://github.com/GokulSelvaraj-Scientist/tcga-lung-cancer-analysis)

Comprehensive multi-omics analysis of lung adenocarcinoma integrating RNA-seq, somatic mutations, and clinical data from real TCGA patients.

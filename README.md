## Featured Projects

### 🧬 [Gene Expression Analysis](https://github.com/GokulSelvaraj-Scientist/gene-expression-analysis)
Differential expression analysis of RNA-seq data from dexamethasone-treated airway smooth muscle cells using DESeq2. Includes volcano plots, heatmaps, and MA plots.

### 📈 [Survival Analysis](https://github.com/GokulSelvaraj-Scientist/survival-analysis)
Kaplan-Meier survival curves and Cox proportional hazards regression on lung cancer clinical data. Identifies key predictors of patient survival outcomes.

### 🦠 [Microbiome Diversity Analysis](https://github.com/GokulSelvaraj-Scientist/microbiome-diversity-analysis)
Alpha and beta diversity analysis of gut microbiome data across healthy, IBD, and antibiotic-treated groups. Includes PCoA, PERMANOVA, and abundance profiling.
